# 📌 The Problem

Over **1 billion people worldwide live with disabilities**, and millions of neurodiverse individuals—including people with **ADHD, Autism, Dyslexia, Dysgraphia, and Anxiety Disorders**—face digital experiences that are not designed for their cognitive needs.

Traditional applications often present:

- Complex interfaces
- Information overload
- Limited personalization
- Poor accessibility support
- Lack of emotional guidance

These barriers reduce independence, confidence, and digital inclusion.

---

# 💡 Our Solution

**Pravi** is an AI-powered accessibility platform that acts as a personalized digital companion for neurodiverse users.

Rather than expecting people to adapt to technology, **Pravi adapts technology to people.**

Powered by **Google Gemini AI**, the platform provides intelligent assistance, voice interaction, personalized planning, emotional support, and accessibility-first experiences to help users navigate everyday life more confidently.

# 🚀 Getting Started

## Prerequisites

- Flutter SDK
- Android Studio
- Firebase CLI
- Dart SDK

---

## Clone Repository

```bash
git clone https://github.com/ShrutiThakur1729/Pravi.git

cd Pravi
```

---

## Install Dependencies

```bash
flutter pub get
```

---

## Configure Firebase

Create a Firebase project and add your configuration files.

Android:

```
android/app/google-services.json
```

iOS:

```
ios/Runner/GoogleService-Info.plist
```

---

## Run Application

```bash
flutter run
```
